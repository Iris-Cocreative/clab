Static HTML prototype for **clab.iriscocreative.com**, hosted via GitHub Pages.

## Structure

| Page | File |
|---|---|
| Home | `index.html` |
| About | `about.html` |
| The Work | `the-work.html` |
| Community | `community.html` |
| Get Involved | `get-involved.html` |

## Design system

- Tokens, typography, components: `assets/css/styles.css`
- Nav + reveal interactions: `assets/js/main.js`

**Typography:** Fraunces (display serif) + DM Sans (body) via Google Fonts.
**Color philosophy:** Warm earth tones as base, prismatic rainbow accents used sparingly.
**Motion:** Slow, organic, respects `prefers-reduced-motion`.
